Getting Started

Installation

1

Install Docker

Make sure Docker is running on your system. Leviathan supports Linux, macOS, and Windows with WSL2.

SHELL
# Verify Docker is running
$ docker --version
Docker version 24.x.x, build xxxxxxx
2

Pull the Image

Pull the latest Leviathan image directly from Docker Hub.

SHELL
$ docker pull l1m4/leviathan:latest
Status: Downloaded newer image for l1m4/leviathan:latest
3

Run the Container

Launch Leviathan in interactive mode with network host access.

SHELL
$ docker run -it --rm --network host l1m4/leviathan:latest
root@leviathan:~#

How to Use

Usage Guide

▶

Mount a Local Volume

Share files between host and container for results, wordlists, or target data.

SHELL
$ docker run -it --rm --network host \
  -v $(pwd)/workspace:/root/workspace \
  l1m4/leviathan:latest
▶

Privileged Mode

For tools requiring raw socket access or kernel-level interaction.

SHELL
$ docker run -it --rm --privileged --network host \
  l1m4/leviathan:latest
▶

Clone & Build Locally

Customize your own Leviathan build with additional tools.

SHELL
$ git clone https://github.com/flfelipelima/leviathanOffSec
$ cd leviathanOffSec && docker build -t leviathan-custom .
